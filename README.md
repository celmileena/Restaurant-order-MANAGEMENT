🍽️ Restaurant Order Management System

A web-based restaurant management system that allows customers to place orders via QR codes, and provides updates to kitchen and Customer. Built with Django, integrated with Twilio for whatsapp notifications showing order confirmation, and supports media uploads for menu items.

🌟 Features

QR-based Customer Ordering:
Customers scan a QR code at the table and access the menu instantly. They can place orders without calling staff.

Table & Order Management:
 Staff can see active orders per table, update order status.

Real-Time Updates:

Kitchen receives immediate notifications of new orders with a notification sound using mp3.

Staff and customer views are updated for order status changes.

Menu Management:
Add, edit, or delete menu items with images and descriptions.

User Authentication:
Different roles for Admin, Staff, and Kitchen.

Twilio Integration:
Sends notifications for order confirmations.

🛠️ Tech Stack

Backend: Python, Django

Frontend: HTML, CSS, JS, Bootstrap

Database: SQLite (default)

Tools: Git, GitHub, Twilio API

IDE: PyCharm
